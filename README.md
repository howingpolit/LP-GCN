This is the code for our research paper: Lossless and Privacy-Preserving Graph Convolution Network for Federated Item Recommendation.
The folder Exp - EquivProof_Code contains code implementations for equivalence proof verification experiments.
The folder Exp_CommCost_ConvClientLoad holds code and experimental data for investigating communication overhead and convolutional client load.
The folder Exp_FalseItems_AddCommCost includes code and results related to analyzing false items and additional communication overhead.
